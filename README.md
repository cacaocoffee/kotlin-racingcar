# 자동차 경주 게임
## 입력
- [ ] 자동차 이름 입력 
  - [ ] ,로 나누기 
  - [ ] 예외처리 
    - [ ] 이름은 5자 초과 금지
    - [ ] 2대 이상 입력 
- [ ] 시도 횟수 입력
  - [ ] 예외처리
    - [ ] 정수 이외에 입력 
## 출력 
-  [ ] 자동차 이름 출력 문구 
-  [ ] 시도 횟수 출력 문구
-  [ ] 각 자동차 별 전진 횟수 문구 
-  [ ] 최종 우승자 문구

## 로직 
-  [ ] 자동차 객체 생성
  - [ ] 자동차 이동 
- [ ] 최종 우승자 선별
